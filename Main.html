Major Projects, Internships and Experiences

  • Smart Farming of Cotton using Computer Vision and Artificial Intelligence

    Smart Farming of Cotton using Computer Vision and Artificial Intelligence which was done as a project course at IIT Gandhinagar. The project involved the use of Computer Vision and Machine Learning to detect the diseases, analyse the growth, and provide a digital inteerface for the farmers to monitor the growth of the crops. The project was done in collaboration with the L&T Technology Solutions and Department of Artificial Intelligence and Plant Biology at IIT Gandhinagar. The publication is under review.

  • Landslide Susceptibility Mapping using Machine Learning, MATLAB Excellence in Innovation

    Landslide Susceptibility Mapping using Machine Learning, which was done as a project from MATLAB Excellence in Innovation. The project involved the use of Machine Learning and MATLAB to predict the landslide susceptibility of the regions based on the topographical data. The project was done using MATLAB and I was also awarded the Mathworks Sustainability Challenge Award for the same.

  • Exploration of Pollen Biology using Scanning Electron Microscopy and Computer Vision

    Exploration of Pollen Biology using Scanning Electron Microscopy and Computer Vision which was done as an SRIP project at IIT Gandhinagar. The project involved the use of images from Scanning Electron Microscopy to detect the pollen grains and analyse the pollen grains using Computer Vision and Machine Learning. A highly scalable web application was also developed for the same using MERN stack. The model was also deployed on the web using TensorFlowJS. The publication is under review.

  • British Airways Data Science Job Simulation

    Completed a Job simulation on Forage focussing on how data science is a critical component of British Airways' success. Scraped and analysed customer review data to uncover findings. Built a predictive model to understand factors that influence buying behaviour.

    Analysed the impact of Sea Level Rise on the Andaman and Nicobar Islands using Python and Machine Learning. The project involved the use of Machine Learning and Python to predict the impact of Sea Level Rise on the Andaman and Nicobar Islands. The project was done in collaboration with the Department of Earth Sciences at IIT Gandhinagar. A streamlit was also developed for the interactive Visualisation of the results. The publication is under process.

  • CNN implementation on FPGA

    Implemented a Convolutional Neural Network on FPGA using Python and Verilog. The project was done as a part of HackRush 2024 Hackathon at IIT Gandhinagar. I was also awarded the Best Project Award for the same in the computer architecture section.
